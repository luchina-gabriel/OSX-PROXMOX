# OSX-PROXMOX - Run macOS on ANY Computer - AMD & Intel

Install Proxmox VE v7.02 - Next, Next & Finish (NNF).

Open Proxmox Web Console -> Datacenter > NAME OF YOUR HOST > Shell.

Copy, paste and execute.
